# 主页重新设计说明

## 设计概述
根据用户需求，将主页重新设计为三个主要部分，突出社会实践项目的特色，营造更加专业和吸引人的用户体验。

## 新的布局结构

### 第一部分：实践足迹（上方占比大）
**位置**：页面顶部，占据主要视觉空间
**内容**：
- **主标题**：实践足迹
- **副标题**：记录每一次实践的精彩瞬间与深刻感悟
- **内容布局**：采用网格布局，左侧大卡片 + 右侧两个小卡片

#### 左侧大卡片 - 实践见闻
- **图标**：层次结构图标，象征实践的深度和广度
- **标题**：实践见闻
- **描述**：深入基层，亲身体验，记录真实的社会实践故事
- **统计数据**：
  - 走访地点：15个
  - 访谈对象：32人
  - 实践天数：28天

#### 右侧小卡片
- **调研成果**：收集一手资料，形成详实调研报告
- **实践日志**：记录每日实践心得与感悟

**设计特点**：
- 左侧大卡片占据2/3宽度，右侧两个小卡片各占1/3
- 响应式设计，在小屏幕上自动调整为单列布局
- 江南水乡风格的半透明背景和优雅边框

### 第二部分：四个小部分（点击进入）
**位置**：实践足迹下方
**标题**：实践探索
**内容**：四个功能模块，每个都可以点击进入相应页面

#### 功能模块
1. **实践感想**
   - 图标：对勾圆圈图标
   - 描述：记录实践过程中的深刻感悟与思考，分享个人成长与收获

2. **实践总结**
   - 图标：文档图标
   - 描述：系统梳理实践成果，总结经验教训，为今后的实践活动提供指导

3. **实践方法**
   - 图标：层次结构图标
   - 描述：分享实践中的有效方法与技巧，为其他实践者提供参考

4. **团队风采**
   - 图标：团队图标
   - 描述：展现团队成员的精彩瞬间，记录团队协作的温暖故事

**设计特点**：
- 2x2网格布局，每个卡片大小一致
- 悬停效果：上移、边框加深、阴影增强
- 每个卡片都有独特的图标和描述

### 第三部分：交互式地图
**位置**：页面底部
**标题**：实践足迹地图
**副标题**：点击地图标记，探索我们的实践足迹
**功能**：集成MapView组件，展示实践地点和路线

## 技术实现

### 1. 组件化设计
- 使用Vue 3 Composition API
- 图标组件化，便于维护和复用
- 响应式布局，适配各种屏幕尺寸

### 2. 样式系统
- 保持江南水乡主题色彩
- 使用CSS Grid和Flexbox实现复杂布局
- 半透明背景和毛玻璃效果
- 优雅的动画和过渡效果

### 3. 交互设计
- 卡片悬停效果
- 点击导航功能
- 响应式适配

## 色彩和视觉风格

### 主色调
- **深青色** (#2D5A4A)：标题和重要文字
- **青色** (#4A9B8C)：描述文字和图标
- **水色** (#B8E6E1)：背景和边框

### 视觉效果
- 半透明背景营造水乡朦胧感
- 渐变色彩增强层次感
- 柔和阴影营造立体感
- 优雅的悬停动画

## 响应式设计

### 大屏幕 (1024px+)
- 实践足迹：2fr 1fr 1fr 三列布局
- 四个小部分：2x2网格布局
- 地图：完整宽度显示

### 中等屏幕 (768px-1024px)
- 实践足迹：单列布局
- 四个小部分：2x2网格布局
- 地图：完整宽度显示

### 小屏幕 (<768px)
- 实践足迹：单列布局
- 四个小部分：单列布局
- 地图：完整宽度显示
- 字体大小和间距适当调整

## 用户体验优化

### 1. 信息层次
- 实践足迹作为主要内容，占据视觉焦点
- 四个功能模块清晰分类，便于用户理解
- 地图作为辅助信息，展示实践范围

### 2. 交互反馈
- 悬停效果提供视觉反馈
- 点击区域明确，操作直观
- 动画效果增强用户体验

### 3. 内容组织
- 从宏观到微观的信息架构
- 实践足迹 → 具体功能 → 地理展示
- 符合用户认知习惯

## 总结

本次重新设计成功实现了用户的需求：

1. **结构清晰**：三个部分层次分明，功能明确
2. **重点突出**：实践足迹占据主要视觉空间
3. **功能完整**：四个功能模块覆盖实践项目的各个方面
4. **视觉统一**：保持江南水乡风格，提升品牌一致性
5. **响应式友好**：适配各种设备，提供良好的用户体验

新的设计既保持了原有的江南水乡美学特色，又大大提升了页面的功能性和用户体验，真正做到了美观与实用的完美结合。

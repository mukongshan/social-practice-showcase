# 江南水乡风格改造说明

## 项目概述
本次改造将社会实践项目展示网页的前端风格从深色科技风格完全改为江南水乡的青色、绿色风格，营造清新淡雅的水乡氛围。

## 设计理念

### 色彩主题
- **主要青色 (#4A9B8C)**: 江南水乡主色调，代表水乡的宁静与优雅
- **次要绿色 (#7FB069)**: 江南春色，代表生机与希望
- **强调色 (#A8D5BA)**: 淡雅青绿，营造清新感
- **浅色 (#D4E9D7)**: 清新淡雅，增强视觉层次
- **深色 (#2D5A4A)**: 深沉稳重，提供对比度

### 辅助色彩
- **水色 (#B8E6E1)**: 江南水乡水色，营造水乡氛围
- **竹色 (#9DC183)**: 江南竹林，增加自然感
- **荷色 (#E8F4F8)**: 江南荷花，营造诗意感
- **石色 (#C8D5B9)**: 江南石桥，增加古朴感

## 改造内容

### 1. 基础色彩变量 (base.css)
- 新增江南水乡主题色彩变量
- 定义主渐变、水色渐变、深色渐变等
- 为后续样式改造提供统一的色彩标准

### 2. 整体布局 (App.vue)
- **背景**: 从深色科技渐变改为水色到青绿的江南水乡渐变
- **导航栏**: 半透明水色背景，淡雅青色边框
- **按钮**: 淡雅青绿渐变背景，深青色文字
- **标题**: 青色到绿色的渐变效果
- **动画**: 江南水乡风格的发光和脉冲效果

### 3. 主页内容 (HomeView.vue)
- **整体背景**: 江南水乡主题渐变背景
- **标题区域**: 淡雅青色光晕背景，青色到绿色渐变标题
- **功能卡片**: 半透明水色背景，淡雅青色边框
- **图标设计**: 青绿渐变背景，青色数字标识
- **文字色彩**: 深青色标题，青色描述文字
- **悬停效果**: 水乡风格的阴影和发光效果

### 4. 地图组件 (MapView.vue)
- **边框**: 淡雅青色边框，营造水乡氛围
- **阴影**: 淡雅青色阴影，增强立体感
- **背景**: 水色背景，增强水乡特色

## 使用说明

### 1. 色彩变量
```css
/* 主要色彩 */
--jiangnan-primary: #4A9B8C;      /* 主要青色 */
--jiangnan-secondary: #7FB069;    /* 次要绿色 */
--jiangnan-accent: #A8D5BA;       /* 强调色 */

/* 渐变色彩 */
--jiangnan-gradient-1: linear-gradient(135deg, #4A9B8C 0%, #7FB069 100%);
```

### 2. 样式应用
- 背景使用江南水乡渐变
- 边框使用淡雅青色
- 文字使用深青色或青色
- 阴影使用淡雅青色
